# Tolga Cebeci — Personal Website

Personal website for portfolio display, privacy policies, and ad network verification.

## Pages

- **index.html** — Portfolio/home page showcasing projects and personal info
- **privacy.html** — Privacy policy page for mobile apps
- **app-ads.txt** — Ad network authorization file (Facebook, Google)

## Structure

```
/
├── index.html          # Main portfolio page
├── privacy.html        # App privacy policy
├── app-ads.txt         # Authorized ad sellers (IAB standard)
├── css/                # Stylesheets (Bootstrap, FontAwesome, Slick, etc.)
├── js/                 # Scripts (Bootstrap, jQuery, custom logic)
├── images/             # Images and app icons
└── fonts/              # Icon fonts and SVG assets
```
